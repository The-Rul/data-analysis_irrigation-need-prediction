# Irrigation Need Prediction

## Context

This project is based on the Kaggle competition [Playground Series - Season 6, Episode 4](https://www.kaggle.com/competitions/playground-series-s6e4/overview). The main objective of the challenge is to predict a crop's irrigation needs from a set of environmental and agricultural features.

Beyond the competition itself, this repository is also a personal learning project focused on practicing the full data workflow from a junior data perspective: data exploration, preprocessing, feature analysis, model training, and result interpretation. 

It is an opportunity to explore tools as Streamlit to display the results.
